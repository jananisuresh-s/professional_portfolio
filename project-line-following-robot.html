Line-Following Robot

Autonomous Robotics  /  Embedded Systems

An autonomous robot that detects and follows a predefined path using infrared sensors, correcting its trajectory in real time through differential motor control.

Contact me

Why I built this

This was early hands-on work in closed-loop control — a robot that has to sense its environment continuously and correct its own behavior in real time, with no manual input once it starts moving. Line-following is a classic embedded systems project, but it's a genuinely useful foundation: the same sense-decide-actuate loop shows up in industrial automation, warehouse logistics, and automated guided vehicles.

How it works

An IR sensor array continuously reads the surface beneath the robot. Dark and light surfaces reflect infrared light differently, so each sensor returns a distinct signal depending on whether it's over the line or the background. The Arduino reads this array and determines the robot's position relative to the line, then adjusts the left and right motor speeds independently — differential steering — to steer back toward center whenever it starts to drift.

My contributions

  • Designed the complete hardware circuit connecting sensors, driver, and motors
  • Interfaced the IR sensor array with the Arduino and read raw sensor states
  • Wrote the control logic mapping sensor readings to differential motor speeds
  • Integrated the L298N motor driver with the DC motors
  • Tested and tuned sensor threshold values for reliable line detection

Challenges

  • Sensor readings shifted under different ambient lighting, requiring threshold recalibration
  • Getting turning response tight enough to avoid overshooting on sharper curves
  • Reducing electrical noise from the motors affecting sensor readings

Future scope

  • Replace the current threshold-based correction with a PID control loop for smoother path tracking
  • Add support for intersections and branching paths, not just a single continuous line
  • Move from a fixed-speed correction to a speed profile that slows into sharp turns
